Project Overview
This project aims to predict whether a customer will leave (churn) a company or not using Machine Learning techniques.
It helps businesses take proactive actions to retain customers.

🎯 Objective
Analyze customer data
Identify patterns behind churn
Build a predictive model
Provide business insights
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib & Seaborn
Scikit-learn
Jupyter Notebook

📊 Dataset Description
The dataset contains customer details such as:
Gender
Tenure
Monthly Charges
Contract Type
Payment Method
Internet Service
Churn (Target Variable)

🔍 Exploratory Data Analysis (EDA)
Checked missing values
Analyzed data distribution
Visualized churn patterns
Correlation heatmap
Feature relationships

Key Insights:
Customers with high monthly charges are more likely to churn
Long-term customers are less likely to churn
Contract type has strong impact on churn

⚙️ Data Preprocessing
Removed missing values
Converted categorical data using encoding
Feature scaling using StandardScaler
Split dataset into training and testing sets

🤖 Model Building
Algorithm Used: Random Forest Classifier
Train-Test Split: 80% - 20%
Model trained on processed dataset

📈 Model Evaluation
Accuracy Score
Confusion Matrix
Classification Report
